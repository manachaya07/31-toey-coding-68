# 31-toey-coding68
ศึกษาเรื่องเขียนโปรแกรมเพื่อเพิ่มทักษะ 
project name : โปรแกรมเพื่อการนอนที่ดี
โปรแกรมนี้พัฒนาขึ้นเพื่อแก้ปัญหาการนอนดึกและพักผ่อนไม่เพียงพอของวัยรุ่น โดยช่วยคำนวณชั่วโมงการนอนและส่งเสริมวินัยในการเข้านอนให้เหมาะสมต่อสุขภาพและการเรียน
1 ที่มาและความสำคัญ
ปัญหา : นอนไม่พอหยุดจับโทรศัพย์ไม่ได้
เป้าหมาย : กาารนอนที่พอ เมื่อถึงเวลานอน
2 ความสามารถของโปรแกรม
โปรแกรมสามารถรับข้อมูลชื่อ เวลานอน และเวลาตื่น 
คำนวณชั่วโมงการนอนด้วยสูตร (24 − เวลานอน) + เวลาตื่น
แจ้งเตือนเมื่อนอนไม่ถึง 8 ชั่วโมง 
และจำลองการล็อกหน้าจอเมื่อถึงเวลานอนที่กำหนด
3 เครื่องมือเครื่องใช้
พัฒนาโดยใช้ภาษา Python ร่วมกับไลบรารี ipywidgets, IPython.display, datetime และ time บนแพลตฟอร์ม Google Colab
5:วิธีการใช้งาน
เปิดโปรแกรมผ่าน Google Colab กรอกข้อมูลที่กำหนด แล้วกดปุ่มเริ่มทำงาน ระบบจะแสดงผลการคำนวณและสถานะการแจ้งเตือนทันที
